PriceFormatter

The price formatter is a pure function that accepts a Price object and returns a PriceViewModel — a flat set of pre-formatted, ready-to-render primitives. Instead of assembling price strings throughout the app, import the formatter once and destructure the values you need.

Quick start
import { formatPrice } from '@offerlab/pricing';

const vm = formatPrice(price);

// Destructure only the primitives you need
const { formattedAmountShort, shortname, badges } = vm;

Input: Price

The Price interface defines every attribute the formatter can accept. Not all fields are present on every price — the combination depends on the price type.

TypeScript
interface Interval {
  value: number;
  unit: 'day' | 'week' | 'month' | 'year';
}

interface Price {
  amount: number;                          // Price in dollars. 0 → "Free"
  compareAtAmount?: number | null;          // Strikethrough original price
  shortname?: string | null;               // Custom tier label. Auto-generated if null
  billingInterval?: Interval | null;        // Charge frequency (digital subs)
  deliveryInterval?: Interval | null;       // Ship frequency (e-commerce subs)
  shippingAmount?: number | null;           // Shipping cost. 0 → "Free shipping"
  trialDays?: number | null;               // Free trial days before first charge
  oneTimeFeeAmount?: number | null;         // Upfront fee on first billing cycle
  installmentCount?: number | null;         // Number of payments (payment plans)
}
AttributeTypeDescription
amountnumberThe price value in dollars. 0 renders as "Free".
compareAtAmountnumber | nullOriginal price for showing a discount. Rendered with strikethrough.
shortnamestring | nullCustom label for the price tier. Auto-generated from price type when not set.
billingIntervalInterval | nullCharge frequency for digital/service subscriptions.
deliveryIntervalInterval | nullShipment frequency for e-commerce subscriptions.
shippingAmountnumber | nullShipping cost. 0 renders as "Free shipping".
trialDaysnumber | nullFree trial period before billing begins.
oneTimeFeeAmountnumber | nullUpfront fee charged with the first billing cycle.
installmentCountnumber | nullNumber of payments for payment plans.

Output: PriceViewModel

The formatter returns a flat object with every primitive pre-computed. Components destructure only the fields they need — no formatting logic at the call site.

TypeScript
interface PriceViewModel {
  // Amount primitives
  formattedAmount: string;                  // "$49.99"
  formattedAmountShort: string;             // "$9.99 / mo"
  formattedAmountFull: string;              // "$9.99 / month"
  formattedAmountWithInstallments: string; // "$75×3 / mo"
  formattedCompareAt: string | null;        // "$12.99" (strikethrough)

  // Label primitives
  shortname: string;                         // "Subscribe & save"
  priceTypeIcon: IconRef | null;             // arrows-repeat | calendar-days | null

  // Badge primitives
  shippingBadge: string | null;              // "$5.99 shipping" | "Free shipping"
  trialBadge: string | null;                 // "14-day free trial"
  oneTimeFeeBadge: string | null;            // "$25.00 one-time fee"
  badges: string[];                           // All badges in display order
}

Amount primitives

Each amount primitive is tailored for a specific context. The formatter handles all edge cases — decimals, free pricing, installment rounding — so components never need to format numbers themselves.

formattedAmount string

Standalone price display without interval context. Use for inline mentions, cart line items, or anywhere the billing cycle is shown separately.

$49.99
Usage
const { formattedAmount } = formatPrice(price);
// → "$49.99"
// → "Free" when amount is 0
formattedAmountShort string

Compact amount with abbreviated interval suffix. Use in cards, lists, table cells, and other space-constrained placements.

$9.99 / mo
Usage
const { formattedAmountShort } = formatPrice(price);
// → "$9.99 / mo"
// → "$29.99 / yr"
// → "$49.99" (no suffix for one-time prices)
formattedAmountFull string

Full amount with unabbreviated interval. Use in detail views, summaries, offer detail rows, and anywhere space allows.

$9.99 / month
Usage
const { formattedAmountFull } = formatPrice(price);
// → "$9.99 / month"
// → "$29.99 / year"
formattedAmountWithInstallments string

Payment plan format showing per-installment amount and count. Whole dollar amounts drop decimals; fractional amounts keep two decimal places.

$75×3 / mo
Usage
const { formattedAmountWithInstallments } = formatPrice(price);
// → "$75×3 / mo"    (whole dollars: no decimals)
// → "$66.33×3 / mo" (fractional: keeps .00)
formattedCompareAt string | null

The compare-at (original) price, rendered with strikethrough styling. Returns null when no compare-at amount is set.

$12.99
Usage
const { formattedCompareAt } = formatPrice(price);

{formattedCompareAt && (
  <Text style={styles.compareAt}>{formattedCompareAt}</Text>
)}

Label primitives

Labels identify the price tier. Every price always has a shortname — either custom-set by the merchant or auto-generated from the price type.

shortname string

Always present. Returns the merchant's custom shortname if set, otherwise generates one from the price type.

Subscribe & save
Usage
const { shortname, priceTypeIcon } = formatPrice(price);

<View style={styles.shortnameRow}>
  {priceTypeIcon && <Icon name={priceTypeIcon} />}
  <Text>{shortname}</Text>
</View>

Badge primitives

Badges surface secondary pricing details — shipping, trials, and fees. The badges array contains all applicable badges in display order, so components just need to render the array.

badges string[]

Pre-sorted array of all applicable badge strings. Render order: shipping → trial → one-time fee. Empty array when none apply.

$5.99 shipping 14-day trial $49.99 one-time fee
Usage
const { badges } = formatPrice(price);

<View style={styles.badgeRow}>
  {badges.map((label) => (
    <Badge key={label}>{label}</Badge>
  ))}
</View>
shippingBadge string | null

Rendered when shippingAmount is set. Displays "Free shipping" when amount is 0.

Free shipping
Examples
// shippingAmount: 5.99  → "$5.99 shipping"
// shippingAmount: 0     → "Free shipping"
// shippingAmount: null  → null
trialBadge string | null

Rendered when trialDays is set.

14-day free trial
Examples
// trialDays: 14  → "14-day free trial"
// trialDays: 7   → "7-day free trial"
// trialDays: null → null
oneTimeFeeBadge string | null

Rendered when oneTimeFeeAmount is set.

$25.00 one-time fee
Examples
// oneTimeFeeAmount: 25    → "$25.00 one-time fee"
// oneTimeFeeAmount: 49.99 → "$49.99 one-time fee"
// oneTimeFeeAmount: null  → null

Interval reference

Intervals are returned in two formats. Use formattedAmountShort for space-constrained placements and formattedAmountFull when space allows.
